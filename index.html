Php-Rosetta

Php-Rosetta can be used to handle dynamic text in a PHP project. It was originally developed to support easy translation, but it could be used in any situation which requires dynamic text where loading values out of an XML dictionary would be appropriate. It's released under the LGPL. The original developer was Jake Taylor, but it has been modified and uploaded here so that others may use it.

Basic Usage

//Load some xml into a string...
//...
$rosetta = new rosetta(rosetta::makeLanguage($str)); //we need one language at least
echo $rosetta->getString('hello');
